Software Komputer

Perangkat Lunak dalam Sistem Komputer

Software atau perangkat lunak adalah kumpulan instruksi yang memerintahkan komputer untuk melakukan tugas tertentu. Software tidak memiliki bentuk fisik dan berjalan di atas hardware. Tanpa software, hardware hanyalah kumpulan komponen elektronik yang tidak berguna.

Jenis-Jenis Software

Software Sistem

Software sistem adalah perangkat lunak yang mengelola dan mengontrol operasi hardware komputer. Software sistem menyediakan platform untuk menjalankan aplikasi.

  • Sistem Operasi: Windows, macOS, Linux, Android
  • Device Driver: Mengontrol perangkat hardware tertentu
  • Utility: Antivirus, disk cleaner, backup tools
  • Berjalan di latar belakang (background)

Software Aplikasi

Software aplikasi adalah program yang dirancang untuk melakukan tugas spesifik bagi pengguna. Aplikasi berjalan di atas sistem operasi.

  • Produktivitas: Microsoft Office, Google Docs
  • Multimedia: Photoshop, VLC Media Player
  • Komunikasi: Browser, Email Client, Zoom
  • Game: Video game, game edukasi

Software Pemrograman

Software pemrograman adalah alat yang digunakan developer untuk membuat, menguji, dan memelihara aplikasi atau sistem software lainnya.

  • Text Editor: Notepad++, Sublime Text, VS Code
  • Compiler: Menerjemahkan kode ke bahasa mesin
  • Debugger: Mencari dan memperbaiki bug
  • IDE: Integrated Development Environment

Lisensi Software

Berdasarkan hak penggunaan, software dibagi menjadi open source (kode terbuka) dan proprietary (kode tertutup).

  • Open Source: Kode sumber tersedia, bisa dimodifikasi (Linux, Firefox)
  • Proprietary: Kode sumber tertutup, berbayar (Windows, Adobe Photoshop)
  • Freeware: Gratis digunakan tapi kode tertutup
  • Shareware: Versi percobaan dengan fitur terbatas
