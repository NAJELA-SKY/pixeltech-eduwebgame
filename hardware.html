Hardware Komputer

Komponen Fisik dalam Sistem Komputer

Hardware atau perangkat keras adalah komponen fisik dari sebuah sistem komputer yang dapat dilihat dan disentuh. Hardware bekerja sama dengan software untuk menjalankan fungsi komputasi. Tanpa hardware, software tidak dapat beroperasi.

Komponen Utama Hardware

CPU (Central Processing Unit)

CPU adalah otak dari komputer yang menjalankan instruksi dari program komputer. CPU melakukan operasi aritmatika, logika, kontrol, dan input/output.

  • Mengelola semua operasi komputer
  • Memproses data dan instruksi
  • Kecepatan diukur dalam GHz
  • Contoh: Intel Core i7, AMD Ryzen

RAM (Random Access Memory)

RAM adalah memori jangka pendek komputer yang menyimpan data yang sedang aktif digunakan. RAM bersifat volatile (data hilang saat daya dimatikan).

  • Menyimpan data sementara yang sedang diproses
  • Semakin besar RAM, semakin banyak aplikasi yang bisa berjalan bersamaan
  • Diukur dalam GB (Gigabyte)
  • Contoh: DDR4, DDR5

Motherboard

Motherboard adalah papan sirkuit utama yang menghubungkan semua komponen komputer. Semua perangkat keras terhubung ke motherboard.

  • Menghubungkan CPU, RAM, dan komponen lainnya
  • Menyediakan koneksi untuk perangkat eksternal
  • Mengatur aliran data antar komponen
  • Contoh: ATX, Micro-ATX, Mini-ITX

Perangkat Input/Output

Perangkat input digunakan untuk memasukkan data ke komputer, sedangkan perangkat output menampilkan atau menghasilkan hasil dari pemrosesan data.

  • Input: Keyboard, Mouse, Scanner, Microphone
  • Output: Monitor, Printer, Speaker, Projector
  • Input/Output: Touchscreen, Modem, Network Card
  • Menjadi perantara antara pengguna dan komputer
