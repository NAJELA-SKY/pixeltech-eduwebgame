Game Edukasi

Belajar Hardware & Software dengan Cara Menyenangkan

Halaman ini adalah template untuk game edukasi tentang hardware dan software. Di sini, Anda dapat menyisipkan game HTML5, JavaScript, atau jikarame untuk game eksternal. Template ini sudah dilengkapi dengan struktur yang siap digunakan.

Area Game

Game akan ditampilkan di area ini.

Untuk menambahkan game, ganti konten di dalam div dengan class "game-placeholder" dengan game Anda.

Contoh Game Sederhana yang Dapat Ditambahkan:

  • Game mencocokkan komponen hardware dengan fungsinya
  • Game puzzle mengatur urutan booting komputer
  • Game tebak gambar komponen komputer
  • Game memori (memory match) dengan istilah hardware/software

Cara Menambahkan Game ke Halaman Ini:

  1. Buat game menggunakan HTML5, JavaScript, atau framework game seperti Phaser
  2. Simpan file game di folder yang sama dengan website ini
  3. Ganti konten di dalam div dengan class "game-placeholder" dengan kode game Anda
  4. Atau gunakan iframe untuk menampilkan game dari sumber eksternal

Contoh kode iframe untuk game eksternal:

<iframe src="game-anda.html" width="800" height="600" frameborder="0"></iframe>
